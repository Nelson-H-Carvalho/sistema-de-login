# Sistema de Login

Projeto de sistema de autenticação simples utilizando HTML, CSS e JavaScript.

## Funcionalidades

- Cadastro de usuário
- Login
- Validação de dados
- Persistência com LocalStorage
- Área logada (dashboard)

## Tecnologias

- HTML
- CSS
- JavaScript

## Objetivo

Praticar autenticação básica e manipulação de dados no front-end.
